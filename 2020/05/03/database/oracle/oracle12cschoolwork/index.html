实验二

任务一：创建数据表

  1. 创建Book表（图书表）、BookType表（图书类型表）、Reader表（读者表）和ReaderType表（读者类型表）
  2. 创建Borrow表（借阅表）和Publish表（出版社表）
CREATE TABLE BookType (
BT_ID CHAR(2) PRIMARY KEY,
BT_Name VARCHAR2(20),
BT_Info VARCHAR2(50)
);

CREATE TABLE Book (
B_ID CHAR(10),
B_Name VARCHAR2(40),
Author VARCHAR2(20),
BT_ID CHAR(2),
P_ID CHAR(4),
PubDate DATE,
Price NUMBER(5,2),
CONSTRAINT FK_book_booktype FOREIGN KEY(BT_ID) REFERENCES BookType(BT_ID)
);

CREATE TABLE Reader (
R_ID CHAR(10) PRIMARY KEY,
R_Name VARCHAR2(9),
RT_ID CHAR(1),
RDept VARCHAR2(20),
RTel VARCHAR2(11)
);

CREATE TABLE ReaderType (
RT_ID CHAR(1) PRIMARY KEY,
RT_Name VARCHAR2(10),
LimitNum NUMBER,
LimitDays NUMBER
);

CREATE TABLE Borrow (
R_ID CHAR(10),
B_ID CHAR(10),
LendDate DATE,
ReturnDate DATE,
LimitDays NUMBER,
BorrowInfo VARCHAR2(3)
);

CREATE TABLE Publish (
P_ID CHAR(4) PRIMARY KEY,
Pubname VARCHAR2(30),
PubTel VARCHAR2(20)
);

任务二：修改数据表

  1. 将Reader表中的联系电话字段的数据类型修改为VARCHAR2(20)。
  2. 指定Borrow表的借阅日期不允许为空。
  3. 删除Borrow表中的限借天数LimitDays字段。
  4. 为Book表增加ISBN字段，数据类型为VARCHAR2（13）。
ALTER TABLE Reader MODIFY RTel VARCHAR2(20);
ALTER TABLE Borrow MODIFY LendDate NOT NULL;
ALTER TABLE Borrow DROP COLUMN LimitDays;
ALTER TABLE Book ADD ISBN VARCHAR2(13);

任务三：建立约束

  1. 为Book表添加主键，约束名为PK_Book。
  2. 为Borrow表添加主键，主键是RID，B_ID，LendDate，约束名为PK_Borrow。
  3. 设置Book表的出版社编号字段P_ID是外键，参照 Publish表中的 P_ID，约束名为FK_book_pub。
  4. 为Book表中的价格字段Price添加检査约束，要求价格必须大于0，约束名为CHK_price。
ALTER TABLE Book ADD CONSTRAINT PK_Book PRIMARY KEY(B_ID);
ALTER TABLE Borrow ADD CONSTRAINT PK_Borrow PRIMARY KEY(R_ID,B_ID,LendDate);
ALTER TABLE Book ADD CONSTRAINT FK_book_pub FOREIGN KEY(P_ID) REFERENCES Publish(P_ID);
ALTER TABLE Book ADD CONSTRAINT CHK_price CHECK(Price > 0);

实验三

任务一：检索限定与排序

  1. 查询所有图书的基本信息。
    select * from book;
  2. 查询所有的图书编号、名称和价格。
    select b_id, b_name, price from book;
  3. 查询教师一次可以借多少本书以及借书期限多少天，输出的结果字段名分别用汉字“借书本数”和“借书期限”表示。
    select limitnum "借书本书", limitdays "借书期限"
from readertype;
  4. 查询“人民邮电出版社”出版的图书的书名、作者、出版社编号及出版日期。
    select b.b_name,b.author,b.p_id,b.pubdate
from book b,publish p
where p.pubname = '人民邮电出版社' and b.p_id = p.p_id;
  5. 查询姓“张”读者的基本信息。
    select * from reader where r_name like '张%';
  6. 查询Borrow表中未还图书的记录。
    select * from borrow where returndate is null;
  7. 查询2012年一年以内的借阅记录。
    select * from borrow
where lenddate between date '2012-1-1' and date '2012-12-31';
  8. 查询部门在“学生处”或“后勤处”的读者信息。
    select * from reader
where rdept = '学生处' or rdept = '后勤处';
  9. 查询所有图书的图书编号、名称、作者、出版日期和价格，并按价格升序排序，若图书价格相同，则按出版日期降序排序。
    select b_id, author, b_name, pubdate, price
from book
order by price asc, pubdate desc;
  10. 查询“BorrowInfo”不为空的借阅记录，并对检索结果按还书日期升序排序，若还书日期为NULL，则还书日期为NULL的行排在最前面。
    select * from borrow
where borrowinfo is not null
order by returndate nulls first;

任务二：函数

  1. 将书名为“SQL Server应用技术”对应的英文字母全部大写并显示修改后的该图书信息。
    select b_id,upper(b_name),author,bt_id,isbn,p_id,price,pubdate
from book
where b_name='SQL Server应用技术';
  2. 查询读者借书证编号、读者姓名、姓名的长度、姓名中是否包含“丽”（显示“丽”所在的位置）。
  3. 查询一个月内还书的借阅信息。
  4. 查询图书信息，包括图书编号、图书名称、作者、出版日期以及价格，其中价格前置本地货币符号，小数点后保留两位, 并以Price为列标题。
  5. 查询出版日期早于2008年8月8日的图书信息，包括图书编号、图书名称、作者、出版日期以及价格，给定检索日期格式为2008-08-08（YYYY-MM-DD）,显示结果的日期格式为星期X 08/08/08（DAY DD/MM/RR）。
  6. 查询未归还图书的借阅信息，将还书日期全部设置为系统时间，若“BorrowInfo”为空，则改为“是”（更改后的列标题仍为“ReturnDate”和“BorrowInfo”）。
  7. 查询计算机类图书的最低价格、最高价格、平均价格以及总价格。
  8. 统计读者表中教师、职工以及学生的人数并将各个分组的读者姓名显示出来（使用LISTAGG函数：LISTAGG(列名,分隔符) WITHIN GROUP(ORDER BY 列名)）。
  9. 查询各出版社下各类图书的平均价格、最高价格和最低价格，并按出版社顺序升序排序。
  10. 查询平均价格大于30的图书类别及该类别图书的平均价格。

任务三：多表查询

  1. 查询Book表中所有图书的书名、作者、出版社名称以及价格。
    select b.b_name,b.b_name,p.pubname,b.price
from book b,publish p
where b.p_id = p.p_id;
  2. 查询Borrow表中所有借书的读者的借书证号、姓名以及所借图书的书名、作者、出版日期和ISBN。
    select borrow.r_id,reader.r_name,book.b_name,book.author,book.pubdate,book.isbn
from book,borrow,reader
where book.b_id = borrow.b_id and borrow.r_id = reader.r_id;
  3. 查询Book表中所有图书的书名、作者、图书类别名称及价格（使用USING子句）。
    select b.b_name,b.author,bt.bt_name,b.price
from book b inner join booktype bt
using(bt_id);
  4. 查询Borrow表中所有借书的读者的借书证号、姓名以及所借图书的图书编号（使用JOIN…ON子句）。
    select borrow.r_id, reader.r_name, borrow.b_id
from borrow join reader on borrow.r_id = reader.r_id;
  5. 查询Book表中属于“计算机类”、“通信类”或“数字艺术类”图书的书名、作者、图书类别名称、出版社名称、出版日期、价格以及ISBN（使用JOIN…ON子句）。
    select b.b_name, b.author, bt.bt_name, p.pubname, b.pubdate, b.price, b.isbn
from book b join booktype bt on b.bt_id = bt.bt_id
join publish p on b.p_id = p.p_id
where bt.bt_name in ('计算机类','通信类','数字艺术类');
  6. 查询读者的借书证编号、姓名、所在部门以及其借阅图书信息。
    select r.r_id, r.r_name, r.rdept,b.borrowinfo
from reader r, borrow b
where r.r_id = b.r_id;
  7. 查询借阅记录对应的读者信息。
    select distinct r.*
from reader r, borrow b
where r.r_id = b.r_id;

任务四：子查询

  1. 查询所有单价小于平均单价的图书编号、书名及价格。
    select b_id, b_name, price
from book
where price < (select avg(price)from book);
  2. 查询和某个出版社最低价格相同的图书的信息。
  3. 查询比“北京交通大学出版社”价格高的其他任何出版社的图书信息。
    select *
from book
where price > (select b.price
    from book b,publish p
    where b.p_id = p.p_id and p.pubname = '北京交通大学出版社');
  4. 查询与《平面构成设计教程》同一出版社的所有图书的图书名称、作者、ISBN。
    select b_name, author, isbn
from book
where p_id = (select p.p_id
              from book b, publish p
              where b.b_name = '平面构成设计教程' and b.p_id = p.p_id);
  5. 查询姓名为“张丽丽”的读者的借阅记录。
    select *
from borrow
where r_id = ( select distinct r.r_id
                from reader r,borrow b
                where r.r_id = b.r_id and r.r_name = '张丽丽');

任务五：集合运算

  1. 查询借阅了书名中开头含有“财经”2个字的图书，或者借阅了书名中开头含有“系统”2个字的图书的读者姓名。
  2. 查询借阅了书名中开头含有“财经”2个字的图书，并且也借阅了书名中开头含有“系统”2个字的图书的读者姓名。
  3. 查询借阅了书名中开头含有“财经”2个字的图书，但是没有借阅书名中开头含有“系统”2个字的图书的读者姓名。

实验四

任务一：视图

  1. 建立“人民邮电出版社”所出版的图书视图V_Pub，视图中包含图书编号、图书名称、出版社编号及出版日期。
    create view V_Pub
  as
select b.b_id,b.b_name,b.p_id,b.pubdate
from book b,publish p
where p.pubname = '人民邮电出版社';
  2. 创建一个借阅统计视图，名为V_Count_1，包含读者借书证编号和总借阅本数(列别名分别为READER_ID和NUMBERS)，要求该视图只读。
    create view V_Count_1(READER_ID,NUMBERS)
  as
select r_id, count(r_id)
from borrow
group by r_id
with read only;
  3. 创建一个借阅统计视图，名为V_Count_2，包含借阅总本数大于一本的读者借书证编号和总借阅本数(列别名分别为READER_ID和NUMBERS)。
    create view V_Count_2(READER_ID,NUMBERS)
  as
select r_id, count(r_id)
from borrow
group by r_id
having count(r_id) > 1;
  4. 修改视图V_Pub，要求增加图书的单价信息且单价大于28元。
    create or replace view V_Pub
  as
select b.b_id,b.b_name,b.p_id,b.pubdate,b.price
from book b,publish p
where b.p_id = p.p_id and p.pubname = '人民邮电出版社' and b.price > 28;
  5. 删除视图V_Pub。
    drop view V_Pub;

任务二：索引

  1. 为Book图书表的书名列创建唯一索引idx_Bname。
    create unique index idx_Bname
  on book(Bname);
  2. 将索引idx_Bname重命名为index_Bname（重命名索引：ALTER INDEX old_index_name RENAME TO new_index_name）。
    alter index idx_Bname rename to index_Bname;
  3. 删除索引index_Bname。
    drop index index_Bname;
